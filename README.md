# cycling

### Quick start (for development):

The application file system layout is:
```
cycling/
|-- client/        # web app
|-- server/        # RESTful-server
|-- ..
|-- README.md
```

Install dependecies:
```shell
npm install tds -g
# server setup
cd server && npm install && tsd install && cd ..
# client setup
cd client && npm install && tsd install
```
