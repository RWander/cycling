# cycling

### Quick start

The application file system layout is:
```
cycling
├── client        # web app
├── server        # RESTful-server
├── ..
└── README.md
```

Install dependecies:
```shell
npm install gulp bower tds -g
# server setup
cd server && npm install && cd ..
# client setup
cd client && npm install && tsd install && bower install
```
